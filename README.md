# TokenLens Capture

> AI token analyzer for your browser — track, analyze, and improve
> your AI conversations across ChatGPT, v0, Claude, and Gemini.

## What it does

- 📊 Analyze token usage of any AI conversation
- ✨ Improve your prompts with AI (Claude / GPT / Groq)
- 🤖 AI Builder — answer questions, get perfect prompts generated for you
- 🗂️ Project Memory — remembers your stack across sessions
- 💉 Insert improved prompts directly into the chat

## Supported Sites

| Site | Analyze | Improve | AI Builder |
|------|---------|---------|------------|
| ChatGPT | ✅ | ✅ | ✅ |
| v0.dev | ✅ | ✅ | ✅ |
| Claude.ai | ✅ | ✅ | ✅ |
| Gemini | ✅ | ✅ | ✅ |

## Installation

### Option A — Load unpacked (Developer mode)

1. Download or clone this repo
2. Open Chrome → `chrome://extensions`
3. Enable **Developer mode** (toggle, top right)
4. Click **Load unpacked** → select this folder
5. Pin the TokenLens icon to your toolbar

## Setup

1. Click the TokenLens icon on any AI site
2. For AI-powered features, add your API key in the **🤖 AI Builder** tab:
   - [Get Groq key (FREE)](https://console.groq.com/keys) ← recommended
   - [Get Anthropic key](https://console.anthropic.com/settings/keys)
   - [Get OpenAI key](https://platform.openai.com/api-keys)
3. The server URL is pre-configured to the hosted server. To use your own:
   - **Hosted:** `https://tokenlens-mcp-production.up.railway.app`
   - **Local:** `http://localhost:3000` (run [tokenlens-mcp](https://github.com/im43ii/tokenlens-mcp) first)

## Usage

### 📊 Analyze Tab

Open any AI conversation → click the TokenLens icon → **Analyze Conversation**

Returns: total token count, estimated cost, waste item count, and the top optimization tip.

### ✨ Improve Tab

Type or paste any prompt → click **Improve Prompt**

Returns three variations with quality scores:
- **Quick** — concise, 1–2 sentences
- **Pro ⭐** — structured with sections and requirements
- **Detailed** — full file structure, implementation steps, edge cases, and a testing checklist

### 🤖 AI Builder Tab

1. Describe what you want to build in plain language
2. Answer 2–4 smart questions (framework, database, auth, etc.)
3. Get a perfect professional prompt generated by your chosen AI

Supports Claude Haiku, GPT-4o mini, and Groq Llama 3.3 (free tier).

### 🗂️ Project Tab

Save your project name, description, tech stack, and current task once.
TokenLens will:
- Skip questions it already knows the answer to in AI Builder
- Prepend your full project context to every generated prompt
- Track completed tasks and build a prompt history

## Privacy

- Your API keys are stored locally in your browser only — never sent to TokenLens servers
- Conversations are analyzed locally or sent to your own self-hosted server
- No usage data is collected by TokenLens
